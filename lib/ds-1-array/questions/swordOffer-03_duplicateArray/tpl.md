## 确定数组是否有重复的元素

### 题目

在一个长度为 `n` 的数组里的所有数字都在 `0 ~ n-1` 的范围内。数组中某些数字是重复的，但是不知道有几个数字重复，并且不知道重复了几次。
判断数组中是否存在重复的数字

示例:

```
输入: [2, 3, 1, 0, 2, 5, 3]
输出: true

输入: [1, 2, 3, 4, 5]
输出: false
```
